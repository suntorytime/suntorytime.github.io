Life in Structure

by Santa Maria, Steven * July 24, 2015

I feel that no matter how crazy or upsetting our world may seem at times as we read the news, it's comforting that there's always an aspect of structure somewhere even in the littlest things. For example, as I'm sitting in this San Francisco cafe, I see a series of artwork horizontally across.

What's so special (or ordinary) about this? Well, it's an easy way to present art--to put another way, information. To take a step further, let's say I wanted to buy one of artwork; I could refer to the chosen one by a digit in the series (e.g. "I'd like to buy the 1st one from the left"). It's a logical, sane, and easy way to communicate. Arrays in the programming sense function much the same way.

An array is a structured set of data or values connected to an index.

Why are arrays helpful? We can refer to an array as a single variable and we can call for individual members, or element, of an array according to an index. So if you want a specific value, we can call the index.

array = ["This", "is", "an", "array"]
puts array[0]
puts array[1]
puts array[2]
puts array[3]
---OUTPUT---
This
is
an
array

Arrays however can have more than one dimension. A one dimension array is a vector and a two dimension one is a matrix, like the following example:

matrix = [["First","element"],["Second","Element"],["Third", "element"]]

Sometimes, an array data structure doesn't make sense. An English dictionary, for example, wouldn't benefit terribly much from an array. Instead, we could leverage hashes to structure a word-to-definition relationship.

A hash, broadly-speaking, is a data structure for key and value pairs. It's similar to an array in that our index is the key and with respect to a value.

Here is an another example of a hash:

hash = { "ca" => "California",
"tx" => "Texas",
"or" => "Oregon"
}

In this case, if I called on the key "ca", it yields the value "California".

Hashes help us retrieve information even if the key-value table is unordered. That is, if we called on a key, it would yield our value.

As we delve deeper into our education in programming, we begin to see the world differently. Perhaps we begin to see the world in a more systematic and logical sense, which is a perspective that will hopefully yield value for our future customers.
